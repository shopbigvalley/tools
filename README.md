# Big Valley Heating Promo Page

This is a simple static promo page for Big Valley Heating.

## Updating the monthly promotion

1. Replace `static/Promo1.pdf` with the new promotion PDF.
2. Keep the filename exactly `Promo1.pdf`.
3. Commit and push the change to GitHub.

Render will auto-deploy after the push, and the page will load the latest PDF.

## Contact buttons

- `Call Today` opens `604-467-6474`.
- `Contact Us Today` opens an email to `sales@bigvalleyheating.ca` with the subject `Big Valley Promotion`.

## Render

This repository includes `render.yaml` for a Render static site. In Render, create a new Blueprint from:

`https://github.com/shopbigvalley/tools.git`
